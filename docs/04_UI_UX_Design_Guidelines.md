# UI/UX 设计规范与视觉方案 (UI/UX Design Guidelines)

## 1. 核心视觉基调：Flat + 液态玻璃 (Liquid Glass)
**设计目标**：打造极具现代感、通透感且极简的高级日记应用。

为了平衡“阅读的清晰度”与“交互的质感”，迹忆 (TraceMind) 将采用双层视觉架构：
- **底层与内容层（Flat 扁平化）**：所有的背景画布、日记正文排版、卡片底色全部采用极简的扁平化设计。摒弃复杂的投影和渐变，提供最高的文字对比度和最干净的阅读体验。
- **悬浮与交互层（液态玻璃 Liquid Glass）**：所有浮在内容之上的元素（如顶部导航栏、底部操作栏、弹出菜单、右下角的“+”号按钮）全部采用**液态玻璃**材质。
  - *注：液态玻璃的渲染由专门的外部库支持，具备真实的光线折射、高强度背景模糊以及边缘内发光，且交互时会有如水滴般的物理 Q 弹反馈。*

## 2. 浅色模式与深色模式 (Light & Dark Mode)
日记应用必须完美支持日间和夜间环境，系统将提供深浅双色模式，并支持跟随系统自动切换。
- **浅色模式 (Light Mode)**：
  - 扁平层：极浅的冷灰色背景，搭配干净的白色卡片。
  - 玻璃层：呈现清透的磨砂白玻璃质感，透出底部的轻微色彩。
- **深色模式 (Dark Mode)**：
  - 扁平层：深邃的黑灰色（如 OLED 黑或极深灰），降低屏幕刺眼感，适合睡前写日记。
  - 玻璃层：呈现深色亚克力质感，边缘高光更加明显，营造出夜间霓虹般的层次深度。

## 3. 个性化主题色切换 (Dynamic Theme Colors)
**场景**：每个人的日记都有属于自己的色彩情绪，不能被单一的蓝色或绿色局限。
**逻辑**：
- App 将提供一系列精心调配的主题色（如：莫兰迪蓝、森林绿、日落橘、神秘紫等）。
- 用户在“设置”中切换主题色后，整个 App 的视觉焦点会随之改变：
  - 所有的主操作按钮、高亮文字、日历打卡热力图的节点都会变成该主题色。
  - **流体光晕关联**：在底层的扁平画布四角，会有一层极其柔和、缓慢呼吸的“彩色光晕”。这个光晕的颜色会跟随用户选择的“主题色”变换。当顶部的液态玻璃导航栏滑过这些光晕时，会产生非常漂亮的色彩折射效果，极大地提升 App 的高级感。
