# 迹忆（TraceMind）产品需求文档 (PRD)

## 1. 产品定位
**迹忆（TraceMind）** 是一款主打“本地优先（Local First）”的个人日记与记录管理软件。
核心理念是将数据的完全控制权交还给用户，不依赖云端，保障隐私，同时提供媲美现代笔记软件的排版体验和灵活的目录结构。未来还将结合本地化 AI 技术提供智能总结与回顾。

## 2. 核心用户场景
- **日常随记**：随时随地记录生活碎片，支持快速拍照或插入相册图片。
- **系统性学习记录**：通过多级文件夹（如“学习”->“日语”）分类整理学习笔记。
- **工作/兼职打卡**：记录打工过程、复盘工作经验，独立存放在特定文件夹下。
- **情绪与回忆追踪**：未来结合 AI 分析日记情绪、生成总结，帮助用户更好地回顾人生。

## 3. 功能需求拆解

### 3.1 目录与分类管理
- **多级文件夹（无限极或限定层级）**：用户可以创建父文件夹和子文件夹（例如：`主目录/学习/日语`）。
- **灵活归属**：日记必须能够归属到任意文件夹下，或者存放在根目录。
- **标签系统**：除了物理隔离的文件夹，支持扁平化的 Tag（标签）进行维度管理。

### 3.2 沉浸式富文本日记编辑
- **类似 Word 的输入体验**：支持文字排版。
- **图文混排**：在编辑日记时，支持调用系统相机直接拍照插入，或从系统相册选取图片插入。
- **个性化背景**：支持为单篇日记选择特定的背景（纯色或背景图），提升沉浸感。
- **元数据记录**：自动记录创建时间、修改时间，支持记录当前天气、心情等信息。

### 3.3 数据安全与迁移（本地优先）
- **无账号系统**：打开即用，无需注册登录。
- **完整备份与恢复**：支持一键导出为 `.diary` 压缩包（包含 SQLite 数据库及所有音视频图片文件），并支持一键导入恢复。
- **跨端/跨设备迁移**：用户更换手机或卸载重装时，仅需一个 `.diary` 文件即可无损恢复所有数据。

## 4. 后续演进路线（AI 赋能）
在完善基础日记功能后，将逐步引入离线或隐私安全的 AI 辅助功能：
- **AI 摘要**：自动为长篇日记生成 Summary。
- **情感分析**：追踪长期的情绪变化波动。
- **知识图谱/时间线**：串联用户记录的事件，生成人生时间线回忆。
