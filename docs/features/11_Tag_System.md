# 灵活标签与情绪统计 (Tags & Mood Tracking)

## 1. 这是用来干嘛的？
**目标**：打破文件夹的局限，提供上帝视角的自我认知。
“文件夹”是树状的、物理隔离的（一篇游记只能放在“旅游”文件夹里，不能同时放在“美食”里）。但生活是网状的。一个标签（Tag）系统能够把不同文件夹里但具有共同属性的回忆串联起来。而情绪的统计，能帮助用户更好地进行自我觉察（Mindfulness）。

## 2. 核心逻辑与功能表现

### 2.1 `#标签` 的自由穿梭
- **场景**：你在“工作”文件夹里写了一篇日记，但里面记下了一个关于设计的灵感。
- **逻辑**：
  - 用户在输入文字时，直接打出 `#设计灵感` 加一个空格，这几个字立刻高亮变成一个标签。
  - 在日记的详情或者侧边栏，有一个“标签墙”。点击 `#设计灵感` 这个标签，App 会无视文件夹的物理隔离，把所有散落在“生活”、“工作”、“读书”文件夹中带有这个标签的日记，全部聚合在一个列表里供你查阅。

### 2.2 情绪与天气追踪统计
- **场景**：用户想知道自己过去一年，到底是快乐多还是焦虑多。
- **逻辑**：
  - 此前我们在“日记编辑器”中提到，每篇日记会自动或手动记录天气（晴/雨）和心情（开心/平静/焦虑）。
  - App 单独提供一个“数据统计面板（Dashboard）”。
  - 以精美的可视化图表（饼图或折线图）展示用户每月的“心情曲线”。
  - **高阶玩法（标签与情绪的化学反应）**：系统交叉分析标签与情绪，告诉用户：“当你打上 `#熬夜` 标签时，你的情绪 80% 是焦虑的；当你打上 `#撸猫` 时，你的情绪 100% 是开心的”。这种数据能切实帮助用户优化自己的生活方式。
