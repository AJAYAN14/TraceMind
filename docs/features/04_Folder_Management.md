# 文件夹管理功能 (Folder Management)

## 1. 这是用来干嘛的？
**目标**：给用户的日记提供“安身之所”。
人的生活是多维度的。用户可能希望把“学日语的笔记”、“每天打工的感悟”和“旅游的游记”分开管理。这个功能就是为了让用户能像在电脑上管理文件一样，自由创建多个分类目录，并且支持“套娃”（在分类里再建分类）。

## 2. 核心逻辑与功能表现

### 2.1 创建文件夹
- **场景**：用户觉得最近生活多了一个新维度，比如“开始养猫了”，想要专门记录猫咪的成长。
- **逻辑**：用户点击“新建文件夹”，弹出一个输入框，输入名称“我的猫咪”后确认。这个文件夹就会出现在当前的页面列表中。

### 2.2 无限级子文件夹（层级嵌套）
- **场景**：用户的分类非常细致。比如创建了“学习”文件夹，点进去后，又想细分为“英语”和“日语”。
- **逻辑**：允许用户在任何一个文件夹内部，继续创建新的文件夹。系统需要记录每个文件夹的“父级”是谁，从而构建出一个树状图。

### 2.3 导航与定位（我到哪儿了？）
- **场景**：用户点进了很多层文件夹（首页 -> 学习 -> 外语 -> 日语），他需要知道现在具体在哪里，并且能轻松退回去。
- **逻辑**：页面顶部需要提供清晰的“路径指示器（面包屑）”，展示完整的路径名称。用户可以直接点击路径上的“学习”两个字，一键退回到学习这一层。

### 2.4 重命名与删除
- **场景**：用户想改个名字，或者某个维度不再记录了。
- **逻辑**：
  - **重命名**：长按文件夹，选择重命名即可修改名字。
  - **删除**：当用户删除一个文件夹时，必须有明确的安全确认。如果文件夹里面还有日记，逻辑上应该连同里面的日记一并删掉，所以必须弹出红色警告提醒用户“这将删除内部的所有日记”。
