# 桌面小组件与闪念胶囊 (Desktop Widget & Quick Capture)

## 1. 这是用来干嘛的？
**目标**：做到“零阻力”记录。
日记不一定非要正儿八经地坐在桌前写几千字。很多时候，记录的冲动往往只有几秒钟——比如在地铁上听到的一句有趣的话，或者脑海中突然闪过的一个灵感。如果记录的流程是“打开App -> 找文件夹 -> 点新建 -> 写标题 -> 写正文”，用户的记录欲望早就消失了。小组件就是为了截获这些转瞬即逝的念头。

## 2. 核心逻辑与功能表现

### 2.1 桌面快捷输入 (Widget)
- **场景**：无需进入 App 即可记录。
- **逻辑**：
  - 在手机桌面上放置一个带有“液态玻璃”质感的极简小组件（Widget）。
  - 用户点一下桌面上的这个框，直接唤起手机键盘。
  - 快速敲下两句话，点击“发送”（或勾选）。
  - 数据在后台直接静默写入本地数据库。无需强制填写标题，也无需选分类，极致丝滑。

### 2.2 “收件箱”与灵感流
- **场景**：桌面上记下的碎碎念，去了哪里？
- **逻辑**：
  - 这些没有分类的短句子，会被统一归集到 App 内部一个叫“收件箱（Inbox）”或“闪念”的特殊系统目录中。
  - 它们的展示形式不像一篇篇正式的日记，而是像“朋友圈”或“微博”的流式卡片，只显示时间和文字。
  - 用户在周末有空复盘时，可以进入这个“收件箱”，把几条闪念合并成一篇正式的日记，或者把它们归档到具体的“日语学习”、“工作”等文件夹中。
